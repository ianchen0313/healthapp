# 成功逆轉腎 - 慢性腎臟病預測系統

這是一個基於 Streamlit 開發的應用程式，利用機器學習模型預測使用者罹患慢性腎臟病的風險，並提供相關衛教資訊與當地就醫建議。

## 1. 執行 app.py 所需的套件

若您需要在虛擬環境中執行此應用程式，建議先建立虛擬環境後，確保安裝以下 Python 套件：

- `streamlit`: 用於建構網頁應用程式介面
- `pandas`: 用於處理資料與建立 DataFrame 特徵
- `scikit-learn`: 用於建立、訓練機器學習模型 (Logistic Regression, RandomForest 等)
- `Pillow` (程式內引入名稱為 `PIL`): 用於讀取與顯示衛教圖片
- `numpy`: 於模型建構過程中處理數值運算

安裝指令範例：
```bash
pip install streamlit pandas scikit-learn Pillow numpy
```

> **提醒：** 目前資料夾中的 `requirements.txt` 只有記錄到 `scikit-learn` 一個套件，若未來要佈署至雲端平台，建議將上述套件皆補入 `requirements.txt` 中。

---

## 2. 專案框架與資料結構

本專案主要由前端介面 (Streamlit) 與後端模型預測邏輯組成。以下為整體的資料夾結構與各個檔案的功能說明：

```text
healthapp-main/
│
├── app.py                # 應用程式主程式：負責網頁介面呈現、收集使用者輸入資料，並呼叫模型給出預測結果。
├── model.py              # 機器學習模型腳本：定義並訓練預測模型，包含資料前處理 (StandardScaler)、邏輯斯迴歸等演算法，以及特徵欄位順序。
│
├── clean_data.csv        # 清理過後的歷史資料檔 (約 1.3 MB)。
├── clean_data2.csv       # 清理過後的歷史資料檔 (約 2.6 MB，model.py 實際會讀取此檔案來訓練預測模型)。
├── requirements.txt      # 記錄專案的依賴套件 (目前僅有 scikit-learn)。
│
├── 投影片1.jpg ~ 8.jpg     # 關於慢性腎臟病的介紹與相關資訊圖片 (於網頁介面的擴充區塊呈現)。
└── 衛教_1.jpg ~ 7.jpg      # 給予使用者的健康衛教宣導圖片 (於預測結果給出後呈現)。
```

### 系統運作流程：
1. **系統初始化**：當透過 Streamlit 執行 `app.py` 時，程式會先自動匯入 `model.py`，並讀取 `clean_data2.csv` 訓練並產生出 `final_model` 模型物件。
2. **網頁載入**：前端介面會顯示疾病介紹 (投影片圖片)，並展示數十題健康調查問卷 (含年齡、BMI、三高病史及其他症狀等)。
3. **資料預測**：使用者輸入資料並點擊預測後，系統會將特徵轉換為 pandas DataFrame，對齊模型所需的欄位並餵進 Pipeline。
4. **結果呈現**：
   - 如果判定有風險，會顯示紅色警告與患病機率，並根據使用者選擇的地區，推薦該縣市的醫院就診連結。
   - 如果判定健康，會顯示綠色的安全提示。
   - 最終都會呈現相關的衛教資訊圖片 (`衛教_X.jpg`) 給使用者參考。
